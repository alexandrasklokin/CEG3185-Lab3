Open two terminals/commandlines: T1, and T2. 

In both terminals, navigate to the forlder (using cd) where the Java files are located. 
Compile using "javac *.java".

In T1, run the command "java PacketReceiver", and keep open. 
This coorresponds to the server program. 

In T2, run the command "java PacketSender", and keep open. 
Type in the serverIp (including periods), and payload when prompted. 
This coorresponds to the client program.
